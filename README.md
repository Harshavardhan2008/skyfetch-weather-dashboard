# SkyFetch Weather Dashboard ☁️

A real‑time weather dashboard built with vanilla JavaScript, Axios, and the OpenWeatherMap API. Users can search any city, view current conditions, and see a 5‑day forecast. Recent searches are saved using localStorage.

## ✨ Features

- **Current Weather** – Temperature, description, and weather icon.
- **5‑Day Forecast** – Daily predictions in responsive cards.
- **Search** – Look up any city worldwide.
- **Recent Searches** – Quick access to the last five cities (saved in localStorage).
- **Auto‑load** – Last searched city loads automatically on page refresh.
- **Loading States** & **Error Handling** – User‑friendly feedback.
- **Responsive Design** – Works on mobile, tablet, and desktop.

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, Grid, animations)
- Vanilla JavaScript (ES6+)
- [Axios](https://axios-http.com/) for API calls
- [OpenWeatherMap API](https://openweathermap.org/api)
- localStorage for data persistence
